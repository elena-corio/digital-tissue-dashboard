## Visual Rules
- Flat light-grey background
- No gradients in the background
- All richness comes from tissue visuals (color, glow, depth, animation)
- Soft drop shadows, glowing effects, and breathing/pulse animations

## Components
- Buttons: white, rounded, blue active state
- Cards: white, soft shadow, rounded
- Consistent spacing and radius via CSS variables
